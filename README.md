## 📖 Overview

**Wisdom Cultivator AI** is an AI-powered self-reflection journal that helps users organize their thoughts, understand emotions, and encourage personal growth through meaningful conversations.

The application uses **IBM Granite Foundation Models** through **IBM watsonx.ai** to provide intelligent, context-aware responses that guide users toward self-awareness and better decision-making.

This project was developed as part of the **IBM SkillsBuild AI Internship 2026**.

---

# ✨ Features

- 🧠 AI-powered reflection conversations
- 💬 Context-aware responses using IBM Granite
- 📝 Automatic reflection title generation
- 📚 Reflection journal history
- 🔍 Search reflections instantly
- 📊 Reflection statistics dashboard
- ✏️ Rename reflections
- 🗑️ Delete reflections
- 📄 Export reflections as text files
- 🎨 Modern responsive interface
- 🌙 Clean and distraction-free design

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/GaikwadAtharva/Wisdom-Cultivator-AI.git
```

Move into the project folder

```bash
cd Wisdom-Cultivator-AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file and add your IBM credentials.

Example:

```env
API_KEY=YOUR_IBM_API_KEY
PROJECT_ID=YOUR_PROJECT_ID
IBM_URL=https://us-south.ml.cloud.ibm.com
```

Run the application

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
